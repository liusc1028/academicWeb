# Academic Personal Website

This is a professional personal website template for an academic scholar and university faculty member. It features separate pages for introduction, research work, publications, patents, awards, projects, teaching, and academic services. The site uses plain HTML/CSS for classic academic style, and content is managed via Markdown files for easy editing. Placeholder content is provided.

## Structure
- index.html: Home/Introduction
- research.html: Research Work
- publications.html: Publications
- patents.html: Patents
- awards.html: Awards
- projects.html: Projects
- teaching.html: Teaching
- service.html: Academic Services
- /content: Markdown files for each section
- /assets: CSS and images

## Deployment
To deploy on GitHub Pages, push the contents of the `site` folder to your `username.github.io` repository.
